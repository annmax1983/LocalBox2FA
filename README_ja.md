## なぜ LocalBox2FA？

ほとんどの 2FA 認証アプリはスマートフォンやクラウド同期が必要です。LocalBox2FA はブラウザ内で直接動作 — スマホ不要、アカウント不要、データがデバイスから離れることはありません。

| 利点 | 詳細 |
|------|------|
| 🔒 **100% ローカル** | すべての TOTP 計算は Web Crypto API を使用してブラウザ内で完結 |
| 📱 **スマホ不要** | ブラウザのツールバーから直接 2FA コードを生成 |
| 🚫 **アカウント不要** | 登録不要、クラウド同期不要、インストールしてすぐ使用 |
| ⚡ **瞬時に生成** | コード生成は 10ms 未満、ワンクリックでクリップボードにコピー |
| 🌙 **ダークモード** | システムのダークモード設定に自動追従 |
| 🎯 **最小限** | 50KB 未満、フレームワークなし、依存関係なし |

## インストール

1. ブラウザの拡張機能ページを開く：
   - **Chrome**: `chrome://extensions/`
   - **Edge**: `edge://extensions/`
2. 右上の**デベロッパーモード**を有効にする
3. **パッケージ化されていない拡張機能を読み込む**をクリックし、`local-box2-fa` フォルダを選択
4. ツールバーの 🔐 LocalBox2FA アイコンをクリックして開始

---

## 使い方

### コードを生成

1. ツールバーの LocalBox2FA アイコンをクリック
2. TOTP Secret キー（Base32 形式）を入力欄に貼り付け
3. **生成**をクリック — 6 桁のコードと 30 秒のカウントダウンが表示
4. **コピー**をクリックしてクリップボードにコピー

### 履歴に保存

1. コード生成後、**保存**をクリック
2. 名前を入力（例：「GitHub」「Google」）
3. アカウントが履歴リストに表示され、リアルタイムでコードを表示

### 保存済みアカウントの使用

- 保存したすべてのアカウントがリアルタイムでコードとカウントダウンを表示
- 任意のアカウントをクリックしてメイン表示に読み込み
- 30 秒ごとに自動更新

---

## 仕組み

```
Secret キーを入力（Base32）
       ↓
Web Crypto API が HMAC-SHA1 を計算
       ↓
6 桁の TOTP コードを生成（RFC 6238）
       ↓
30 秒カウントダウン + 自動更新
       ↓
ワンクリックでクリップボードにコピー
```

すべての計算はブラウザの Web Crypto API を使用してローカルで完結。ネットワークリクエストなし、データアップロードなし。

## プライバシー

- ✅ **データアップロードなし** — すべての TOTP 計算がローカルで完結
- ✅ **分析・追跡なし** — Cookie、追跡、テレメトリは使用しません
- ✅ **ネットワークリクエストなし** — 拡張機能は完全にオフラインで動作
- ✅ **ローカルストレージのみ** — シークレットはブラウザの localStorage に保存
- ✅ **権限なし** — manifest は権限を宣言しません

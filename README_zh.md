一款轻量级浏览器扩展，在本地生成 TOTP 2FA 验证码 — 零上传，完全离线。

> Chromium 内核 · Manifest V3 · 100% 离线 · 隐私优先

---

## 为什么选择 LocalBox2FA？

大多数 2FA 验证器需要手机 App 或云端同步。LocalBox2FA 直接在浏览器中运行 — 不需要手机，不需要账号，数据永远不会离开你的设备。

| 优势 | 说明 |
|------|------|
| 🔒 **100% 本地** | 所有 TOTP 计算在浏览器中通过 Web Crypto API 完成，零上传 |
| 📱 **不需要手机** | 直接在浏览器工具栏生成 2FA 验证码，无需手机 App |
| 🚫 **不需要账号** | 无需注册、云端同步或登录，安装即用 |
| ⚡ **极速生成** | 验证码生成不到 10ms，一键复制到剪贴板 |
| 🌙 **暗色模式** | 自动跟随系统暗色模式偏好 |
| 🎯 **极简轻量** | 不到 50KB，无框架、无依赖、无臃肿 |

## 安装方法

1. 打开浏览器扩展页面：
   - **Chrome**：`chrome://extensions/`
   - **Edge**：`edge://extensions/`
2. 开启右上角的**开发者模式**
3. 点击**加载已解压的扩展程序**，选择 `local-box2-fa` 文件夹
4. 点击工具栏中的 🔐 LocalBox2FA 图标即可开始使用

---

## 使用方法

### 生成验证码

1. 点击工具栏中的 LocalBox2FA 图标
2. 将 TOTP Secret 密钥（Base32 格式）粘贴到输入框
3. 点击**生成** — 显示 6 位验证码和 30 秒倒计时
4. 点击**复制**将验证码复制到剪贴板

### 保存到记录

1. 生成验证码后，点击**保存**
2. 输入名称（如"GitHub"、"Google"）
3. 账号出现在历史记录列表中，实时显示验证码

### 使用已保存的账号

- 所有保存的账号实时显示验证码和倒计时
- 点击任意账号将其密钥加载到主显示区
- 每 30 秒自动刷新验证码

### 导出备份

- 点击**导出**下载所有保存的密钥为 `.txt` 文件
- 格式：`账号名称:Secret密钥`（每行一条）
- 请妥善保管此文件 — 它包含你的密钥

## 工作原理

```
输入 Secret 密钥（Base32）
       ↓
Web Crypto API 计算 HMAC-SHA1
       ↓
生成 6 位 TOTP 验证码（RFC 6238）
       ↓
30 秒倒计时 + 自动刷新
       ↓
一键复制到剪贴板
```

所有计算都在本地通过浏览器的 Web Crypto API 完成，无网络请求，无数据上传。

## 隐私保护

- ✅ **零数据上传** — 所有 TOTP 计算在本地完成
- ✅ **无分析追踪** — 不使用 Cookie、追踪或遥测
- ✅ **无网络请求** — 扩展完全离线工作
- ✅ **仅本地存储** — 密钥保存在浏览器 localStorage 中，永远不会离开设备
- ✅ **零权限** — manifest 不声明任何权限

---

## 安全说明

- 密钥存储在浏览器 `localStorage` 中 — 会一直保留直到清除浏览器数据
- 清除浏览器数据前请先导出密钥备份
- 扩展不使用 `chrome.storage` — 数据仅限浏览器配置文件
- 为最大化安全，使用后可删除已保存的密钥
